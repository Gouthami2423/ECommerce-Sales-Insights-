E-commerce Sales Insight Dashboard

Interactive dashboard built using Flask, HTML, CSS, and JavaScript.

Features:

Monthly sales trend

Sales by product category

Profit contribution and revenue analysis

Interactive charts with filters

Run Project:
pip install flask pandas
python app.py


Open your browser and go to: http://127.0.0.1:5000# ECommerce-Sales-Insights-


A data analytics dashboard for e-commerce sales, built with Flask, HTML/CSS/JS, and Chart.js, to help businesses visualize sales trends, product performance, and category sales and profit. Helping decision-makers optimize operations and marketing strategies.
